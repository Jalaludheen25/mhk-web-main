# MHK Trading & Ship Chandlers Website

This is the official website for MHK Trading & Ship Chandlers LLC, a leading provider of maritime, industrial, and construction supplies in the UAE.

## Features

*   **Responsive Design:** specific mobile and desktop navigation.
*   **Video Backgrounds:** Dynamic video headers for Home and Products pages.
*   **Product Catalog:** Detailed pages for various product categories (Fresh Provisions, Scaffolding, Logistics, etc.).
*   **Contact Form:** Integrated contact options.

## Technologies Used

*   HTML5
*   Tailwind CSS (via CDN)
*   Vanilla JavaScript
*   AOS (Animate On Scroll) Library
*   Font Awesome Icons

## Deployment

This project involves static HTML files and can be hosted on GitHub Pages, Netlify, Vercel, or any standard web server.
